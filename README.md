## Sobre o Projeto

Montar um README de perfil no GitHub pode ser desafiador, especialmente para quem esta comecando. Este repositorio reune **templates prontos e personalizaveis** que facilitam esse processo, permitindo que qualquer dev tenha um perfil atrativo em poucos minutos.

Cada template esta separado por **estilo visual** e por **area de atuacao** (Backend, Frontend e Fullstack), para que voce encontre exatamente o que precisa.

### Por que usar?

- **Economize tempo** - Templates prontos, so personalizar com suas informacoes
- **Designs variados** - Minimalista, tech, anime, colorido, animado e mais
- **Separado por stack** - Backend, Frontend e Fullstack em cada estilo
- **Facil de usar** - Copie, cole e edite o que precisar
- **Aberto a contribuicoes** - Sugira novos templates ou melhore os existentes

## Como Usar

1. Escolha um template na tabela acima
2. Clique em **Ver** para abrir o template
3. Copie todo o conteudo que esta entre `<!-- INICIO DO TEMPLATE -->` e `<!-- FIM DO TEMPLATE -->`
4. Crie um repositorio no GitHub com o **mesmo nome do seu usuario**
5. Cole o template no arquivo `README.md` desse repositorio
6. Substitua `seu-usuario`, `Seu Nome` e demais placeholders com suas informacoes

## Contribuindo

Contribuicoes sao muito bem-vindas! Se voce tem um template legal ou quer melhorar um existente:

1. Faca um fork do projeto
2. Crie uma branch para sua feature (`git checkout -b feature/novo-template`)
3. Commit suas mudancas (`git commit -m 'Adiciona novo template'`)
4. Push para a branch (`git push origin feature/novo-template`)
5. Abra um Pull Request
